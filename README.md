# ЭлитСтори

Одностраничный сайт для производства мангальных зон под заказ.

## Источники

- Тексты, контакты и позиционирование: бриф `Бриф_разработка_сайта_v4 (4).docx`.
- Все фотографии в `public/assets/photos/`: архив публичной папки Яндекс Диска из задания.
- Визуальные ориентиры: Fuego, Weber Genesis II и OFYR.

Изображения не генерировались и не загружались из внешних фотостоков.

## Запуск

```bash
pnpm install
pnpm dev
```

## Сборка

```bash
pnpm build
```
